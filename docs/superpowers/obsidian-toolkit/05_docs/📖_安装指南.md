# 📖 Obsidian 小说可视化工具包 - 安装指南

> 本指南将帮助你在 Obsidian 中完整安装和配置小说可视化工具包

---

## 📋 前置要求

- **Obsidian** 1.0 或更高版本
- **社区插件**已启用

---

## 🚀 安装步骤

### 第一阶段：安装插件（5分钟）

#### 1.1 启用社区插件

1. 打开 Obsidian
2. 进入 `设置` → `社区插件`
3. 如果提示"安全模式"，点击"关闭安全模式"

#### 1.2 安装必需插件

在社区插件市场中搜索并安装：

| 插件名称 | 必需 | 说明 |
|---------|------|------|
| **Dataview** | ✅ | 查询和索引笔记数据 |
| **Templater** | ✅ | 模板系统和脚本自动化 |
| **Timeline** | ✅ | 时间线可视化渲染 |

#### 1.3 安装推荐插件

| 插件名称 | 推荐 | 说明 |
|---------|------|------|
| **QuickAdd** | ⚡ | 快速创建笔记 |
| **MetaEdit** | ⚡ | YAML 元数据编辑器 |
| **Buttons** | ○ | 可视化更新按钮 |
| **Iconize** | ○ | 图标支持 |

> 💡 **提示**：Mermaid 已内置在 Obsidian 中，无需额外安装

---

### 第二阶段：配置插件（10分钟）

#### 2.1 配置 Dataview

1. 进入 `设置` → `Dataview`
2. 配置选项：
   - ✅ **Enable JavaScript queries**（必需）
   - ✅ **Enable inline JavaScript**（必需）
   - 索引位置：`/`（根目录）

#### 2.2 配置 Templater

1. 进入 `设置` → `Templater`
2. 配置选项：
   - **Template folder**：设置为 `templates`
   - ✅ **Trigger on new file creation**
   - ✅ **Trigger on file modification**
   - **Startup template**：留空

#### 2.3 配置 Timeline

1. 无需特殊配置
2. 安装后即可使用 `type: timeline` 格式

---

### 第三阶段：创建文件夹结构（5分钟）

#### 3.1 手动创建（推荐新手）

在 Obsidian 库根目录创建以下文件夹：

```
你的库/
├── characters/        📁 角色笔记
├── chapters/         📁 章节笔记
├── factions/         📁 阵营笔记
├── plotlines/       📁 情节线笔记
├── templates/        📁 模板文件夹（重要！）
├── scripts/         📁 脚本文件夹
├── visualization/    📁 可视化文件夹
└── metadata/         📁 元数据文件夹
```

#### 3.2 自动创建（使用初始化脚本）

1. 复制 `00_初始化脚本.md` 到 Obsidian 库根目录
2. 在 Obsidian 中打开该笔记
3. 点击运行 Templater 脚本
4. 文件夹将自动创建

---

### 第四阶段：导入模板（5分钟）

#### 4.1 复制模板文件

从工具包复制以下文件到你的 `templates/` 文件夹：

```
templates/
├── 角色模板.md        ← 必需
├── 章节模板.md        ← 必需
├── 阵营模板.md        ← 推荐
└── 情节线模板.md      ← 推荐
```

#### 4.2 配置 Templater 模板

1. 进入 `设置` → `Templater` → `Template settings`
2. 为每个模板设置快捷命令（可选）：
   - `角色模板.md` → `/角色`
   - `章节模板.md` → `/章节`

#### 4.3 复制脚本文件

将脚本文件复制到 `scripts/` 文件夹：

```
scripts/
├── 关系图生成器.md
├── 时间线生成器.md
└── 快速统计面板.md
```

#### 4.4 复制可视化示例

将示例文件复制到 `visualization/` 文件夹：

```
visualization/
├── 01_全局关系图.md
└── 02_全局时间线.md
```

---

### 第五阶段：验证安装（2分钟）

#### 5.1 创建测试角色

1. 按 `Ctrl/Cmd + P` 打开命令面板
2. 输入 `Templater: Create new note from template`
3. 选择 `角色模板.md`
4. 填写角色信息并保存

#### 5.2 创建测试章节

1. 使用命令面板创建章节笔记
2. 选择 `章节模板.md`
3. 填写章节信息

#### 5.3 验证关系图

1. 打开 `visualization/01_全局关系图.md`
2. 确认 Mermaid 图正常渲染

#### 5.4 验证时间线

1. 打开 `visualization/02_全局时间线.md`
2. 确认 Timeline 正常显示

---

## 🔧 故障排除

### 问题1：模板不显示

**原因**：Templater 未正确配置

**解决**：
1. 检查 `设置` → `Templater` → `Template folder` 是否指向正确路径
2. 确保模板文件位于该文件夹中
3. 重启 Obsidian

### 问题2：Dataview 查询失败

**原因**：JavaScript 查询未启用

**解决**：
1. 进入 `设置` → `Dataview`
2. 确保 `Enable JavaScript queries` 已启用
3. 确保文件夹名称与查询中的名称一致

### 问题3：Timeline 不渲染

**原因**：Timeline 插件未安装或格式错误

**解决**：
1. 在社区插件中搜索并安装 "Timeline"
2. 确保第一行是 `type: timeline`
3. 确保使用 ` ```timeline ` 代码块

### 问题4：Mermaid 图不显示

**原因**：代码块格式错误

**解决**：
1. 使用 ` ```mermaid ` 而不是 ` ```graph `
2. 检查 Mermaid 语法是否正确
3. 尝试在 Mermaid Live Editor 中调试
